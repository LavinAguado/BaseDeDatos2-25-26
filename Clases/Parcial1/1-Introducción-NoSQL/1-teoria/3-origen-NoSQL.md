# El origen de NoSQL

NoSQL surge como respuesta a nuevas necesidades tecnológicas que comenzaron a hacerse evidentes con el crecimiento de internet y las aplicaciones globales.

Empresas que manejaban enormes volúmenes de datos, como motores de búsqueda, redes sociales y plataformas de comercio electrónico.

Se necesitaban soluciones capaces de operar de forma distribuida y flexible

- Crecimeinto del Big Data: conjuntos de datos tan masivos, rápidos y complejos que superan la capacidad de las herramientas tradicionales para ser procesados o analizados eficazmente.
- Aplicaciones web masivas y de alto tráfico.
- Uso itensivo de Cloud computing.
- Necesidad de sistemas distribuidos.

**Las 5 V del Big Data:**

* **Volumen** (gran cantidad de datos).
* **Velocidad** (flujo en tiempo real).
* **Variedad** (datos estructurados y no estructurados).
* **Veracidad** (precisión y confianza).
* **Valor** (la información útil extraída).

**NoSQL no significa la eliminación total del modelo relacional SQL. El término se interpreta mejor como “Not Only SQL”, es decir, no solo SQL.**
