# Bases de Datos II – NoSQL, Documentales, Grafos y Vectores

Curso práctico orientado a **bases de datos NoSQL modernas**, con énfasis en:

- Modelado adecuado según el problema
- Consultas reales
- Casos de uso profesionales
- Trabajo continuo en **GitHub**

**Formato del curso**

- Clases teóricas + prácticas
- Entregas mediante repositorio GitHub
- Evaluación basada en proyectos y pruebas prácticas

## Calendario general del curso

### Parcial 1 – Fundamentos NoSQL y BD Documentales

- **Bloque 1:** Introducción a NoSQL
- **Bloque 2:** Bases de Datos Documentales (MongoDB)

### Parcial 2 –BD de grafos y vectoriales

- **Bloque 1:** Bases de Datos de grafos (Neo4j)
- **Bloque 2:** Bases de Datos vectoriales

## Metodología de trabajo

- Cada clase tiene archivos `.md`
- Los diagramas se incluyen como imágenes PNG (Mermaid)
- Las entregas se realizan en repositorios individuales del alumno
- Se valorará:
  - Claridad técnica
  - Justificación de decisiones
  - Organización del repositorio

## Evaluación

- Ejercicios prácticos continuos
- Pruebas prácticas parciales
- Examen final práctico (fuera de las 15 semanas)
